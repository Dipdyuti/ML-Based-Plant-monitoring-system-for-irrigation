# ML-Based-Plant-monitoring-system-for-irrigation
#ML-Based Plant Monitoring System

An intelligent Machine Learning based Plant Monitoring System that monitors environmental and soil conditions using sensors and predicts plant health status. The system helps in smart agriculture by analyzing real-time sensor data and providing recommendations for irrigation and plant care.

---

#Features

- Real-time plant and soil monitoring
- Machine Learning based prediction system
- Soil moisture analysis
- Temperature and humidity monitoring
- pH and rainfall analysis
- Irrigation recommendation system
- Data visualization dashboard
- Accurate prediction using ML models

---

#Machine Learning Models

The following ML models were used and compared:

- Random Forest Classifier
- Decision Tree Classifier
- XGBoost Classifier

The best-performing model was selected based on:
- Accuracy
- Precision
- Recall
- F1-Score

---

#Dataset Features

The dataset contains the following parameters:

- Soil Moisture
- Temperature
- Soil Humidity
- Air Temperature
- Wind Speed
- Air Humidity
- pH
- Rainfall

---

#System Workflow

1. Collect sensor data
2. Store and preprocess data
3. Train machine learning models
4. Predict plant health status
5. Display results on dashboard
6. Generate irrigation recommendations

---

#Output

The system can:
- Predict plant health condition
- Recommend watering requirements
- Detect abnormal environmental conditions
- Visualize sensor readings using graphs
